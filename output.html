Kiro ☘️

Kiro는 텍스트 입력에서부터 기록, 인쇄, 발행에 이르기까지 전 과정을 고려해 설계된 명시적 구조의 마크업 언어입니다.

@link:https://kiro.onlyoli.space ! Ground 에디터 바로 사용해보기 (베타)

✨ 소개

Kiro는 마크다운의 장점을 계승하면서도, 문서의 시각적 구조와 의미 표현을 명시적으로 설계할 수 있도록 만들어진 새로운 마크업 언어입니다. 입력부터 발행까지 흐름이 끊기지 않도록 설계되어 있으며, 복잡한 문법 대신 직관적이고 구조화된 작성 경험을 제공합니다.

이 프로젝트는 Kiro 문법을 렌더링하고 실시간으로 작성할 수 있는 에디터 Ground를 포함하고 있습니다. 자세한 문법 등의 사항은 docs 폴더를 참고해주세요.

📦 기술 스택

  • 프론트엔드: HTML + JavaScript

  • 백엔드: Flask (Python)

  • 렌더러: Kiro Renderer (Python 기반 커스텀 파서)

  • 스타일 시스템: Tailwind CSS

  • 배포: Render 플랫폼

    • 📚 문법 요약

    기본적인 문법 구조는 다음과 같습니다:

  • 대괄호로 감싼 스타일 테그 사용, <> 로 끝내기

  • style 블록을 통해 계층적 스타일, 속성, 폰트, Tailwind 클래스 등을 선언

  • @img:, @video: 등의 접두어로 리소스를 명시적으로 삽입

  • | 기호는 인용문, -, 1.1.은 리스트, >는 토글 기능

  • 마크다운의 대부분 요소는 호환되며, 링크나 이미지는 대체 문법 사용

    • 자세한 문법 설명은 docs/kiro.md 문서를 참고하세요.

    🧠 Kiro의 철학

  • 구조는 스타일보다 먼저 와야 한다

  • 문서의 읽기 경험을 디자인할 수 있어야 한다

  • 가독성과 재사용성을 동시에 만족하는 문법이어야 한다

    • Kiro는 단순한 마크업 언어를 넘어서, <br> 문서 작성 자체에 대한 고민과 철학을 담은 실험입니다.
